# Security

Audit, review, and pen-test skills.

- [deep-review](deep-review/SKILL.md) — verified architectural deep-dive that hunts for code-judo structural simplifications across four sub-agent phases (discover, review, QA, report).
- [red-team](red-team/SKILL.md) — dynamic penetration test against a running web app or API: authenticates as real users and actively attacks live endpoints to prove IDOR/BOLA, privilege escalation, and auth bypass with replayable PoCs.
